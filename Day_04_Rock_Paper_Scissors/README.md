# Day 4 – Rock Paper Scissors

## Overview

This folder contains my solution to the Day 4 project from Angela Yu’s 100 Days of Code: Python Bootcamp.

The project focused on using lists, the `random` module, and conditional statements to build an interactive Rock, Paper, Scissors game.

## Topics Covered

- Lists
- Random Module
- Conditional Statements (`if`, `elif`, `else`)
- User Input
- Game Logic

## What I Learned

- How to store related data using lists
- How to generate random choices using Python's `random` module
- How to compare user input with randomly generated values
- How to implement game logic using conditional statements

## Reflection

Day 4 helped me understand how randomness and decision-making can be combined to create interactive games. It also reinforced the importance of writing clear program logic to handle different outcomes correctly.

## Project Description

The program allows the user to play a game of Rock, Paper, Scissors against the computer.

The game:
- Accepts the user's choice
- Randomly generates the computer's choice
- Compares both choices
- Determines whether the user wins, loses, or draws
