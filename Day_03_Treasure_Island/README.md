# Day 3 – Treasure Island

## Overview

This folder contains my solution to the Day 3 project from Angela Yu’s 100 Days of Code: Python Bootcamp.

The project focused on using conditional statements to create an interactive text-based adventure game where the player's choices determine the outcome.

## Topics Covered

- Conditional Statements (`if`, `elif`, `else`)
- Nested Conditions
- User Input
- String Methods
- Program Flow
- Decision Making

## What I Learned

- How to use conditional statements to control the flow of a program
- How to create nested decision structures
- How to process and compare user input
- How to design a simple interactive text-based game

## Reflection

Day 3 introduced me to the power of conditional logic in Python. Building Treasure Island helped me understand how different decisions can lead to different outcomes and reinforced the importance of organizing program logic effectively.

## Project Description

The game begins at a crossroads where the player must make a series of decisions to reach the treasure.

The player must:
- Choose the correct path
- Decide how to cross the lake
- Select the correct door

Each decision affects the outcome, leading either to victory or a game over.
