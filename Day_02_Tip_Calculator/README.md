# Day 2 – Tip Calculator

## Overview

This folder contains my solution to the Day 2 project from Angela Yu’s 100 Days of Code: Python Bootcamp.

The project focused on performing mathematical calculations, working with different data types, and formatting numerical output to build a simple tip calculator.

## Topics Covered

- Variables
- User Input
- Data Types (`int` and `float`)
- Mathematical Operations
- f-Strings
- Number Formatting

## What I Learned

- How to convert user input into numerical data types
- How to perform arithmetic calculations in Python
- How to calculate percentages and split values between multiple people
- How to format floating-point numbers to display two decimal places

## Reflection

Day 2 helped me become more comfortable working with numbers in Python. 
Building the Tip Calculator demonstrated how programming can be used to solve practical, real-world problems through simple calculations and user interaction.

## Project Description

The program asks the user to enter:
- The total bill amount
- The tip percentage
- The number of people splitting the bill

It then calculates the total amount, including the tip, and displays how much each person should pay, 
formatted to two decimal places.
