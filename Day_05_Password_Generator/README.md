# Day 5 – Password Generator

## Overview

This folder contains my solution to the Day 5 project from Angela Yu’s 100 Days of Code: Python Bootcamp.

The project focused on using lists, loops, and Python's `random` module to build a customizable password generator. It also introduced techniques for manipulating data and converting lists into strings.

## Topics Covered

- Lists
- For Loops
- Random Module
- String Manipulation
- User Input
- List Operations
- Data Conversion

## What I Learned

- How to use lists to store collections of data
- How to iterate through data using `for` loops
- How to generate random values using Python's `random` module
- How to combine letters, numbers, and symbols to generate secure passwords
- How to shuffle elements in a list to produce randomized output
- How to convert a list of characters into a string using the `join()` method

## Reflection

Day 5 expanded my understanding of working with lists and loops by applying them to a practical project. Building the Password Generator showed me how multiple programming concepts can be combined to create a useful application while reinforcing the importance of writing clean and organized code.

## Project Description

The program generates a random password based on the user's preferences.

The user is asked to specify:
- The number of letters
- The number of symbols
- The number of numbers

The program then creates a randomized password by combining the selected characters, shuffling them, and converting the final list into a single string.
